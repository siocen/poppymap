Poppy Playtime Addon of the new mob Huggy Wuggy, it will be a famous character in the horror game Poppy Playtime. The mob behaves just like the original character and looks very similar. The mod is really high quality, download and try it.

Installation:

  • Download Mod;
  • Use Minecraft
  • Open the downloaded file to install;
  • Select it in the settings;
  • Be sure to enable the experimental game mode;
  • Have fun!
